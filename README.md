ProgremacionWeb_aprendizaje
Descripción
Este repositorio documenta el progreso y los ejercicios de aprendizaje enfocados en el desarrollo y la programación web
. A través de múltiples proyectos de práctica, este espacio sirve como un portafolio de estudio, destacando principalmente el trabajo con la biblioteca React.
Tecnologías Utilizadas
El código del repositorio está compuesto por los siguientes lenguajes de programación y estilos
:
CSS (53.1%) - Para el diseño y la maquetación de las interfaces.
TypeScript (36.9%) - Para dotar de tipado estático y mayor robustez al código JavaScript.
JavaScript (6.2%) - Para la lógica de programación.
HTML (3.8%) - Para la estructura de las páginas web.
Estructura del Proyecto
El entorno de trabajo está dividido en una serie de prácticas secuenciales y archivos de configuración para su ejecución en entornos de desarrollo
:
Prácticas de React: Contiene 7 carpetas dedicadas a ejercicios específicos (react01PRACTICA, react02PRACTICA, react03PRACTICA, react04PRACTICA, react05PRACTICA, react06PRACTICA y react07PRACTICA)
.
PrograWeb: Carpeta de código adicional o principal del curso
.
Configuración del Entorno: Incluye la carpeta .idea para la configuración del editor de código (como WebStorm o IntelliJ)
.
Gestión de Dependencias: El proyecto utiliza un ecosistema Node.js, evidenciado por los archivos package.json, package-lock.json y la carpeta node_modules
.
